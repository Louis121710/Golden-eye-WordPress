# Download Logos Instructions

## Logo URLs Found:
1. **Golden Eye Security Logo**: https://louis121710.github.io/golden-eye-website-/assets/golden-eye-logo.png
2. **SAIDSA Logo**: https://louis121710.github.io/golden-eye-website-/assets/saidsa-logo.png

## Quick Download Method:

### Option 1: Direct Download
1. Right-click on each link above and "Save As"
2. Save them to: `app/public/wp-content/themes/golden-eye-security/assets/images/`
3. Name them:
   - `golden-eye-logo.png`
   - `saidsa-logo.png`

### Option 2: Using Browser
1. Visit: https://louis121710.github.io/golden-eye-website-/
2. Right-click on the Golden Eye Security logo → Save Image As
3. Right-click on the SAIDSA logo → Save Image As
4. Save both to: `app/public/wp-content/themes/golden-eye-security/assets/images/`

### Option 3: Command Line (if you have curl)
```bash
cd app/public/wp-content/themes/golden-eye-security/assets/images
curl -O https://louis121710.github.io/golden-eye-website-/assets/golden-eye-logo.png
curl -O https://louis121710.github.io/golden-eye-website-/assets/saidsa-logo.png
```

After downloading, the logos will automatically be used in the theme!
